# TYPO3 Extension: media2click

TYPO3 frontend extension to render external video and iframe content with
privacy in mind: User has to click on placeholder to load the actual content.
You can configure individual placeholder content and the option for permanent
activation on a per target host basis.

## Installation:

The recommended way to install media2click is by
using [Composer](https://getcomposer.org):

```bash
composer require amazing/media2click
```
